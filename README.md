# 🎬 VideoCut-CLI

**VideoCut-CLI** is an AI-powered command-line tool designed to help Content Creators automate their entire video workflow: from downloading and editing (trim, caption, watermark) to publishing on social media platforms.

---

## 🚀 Project Overview

- **Core Stack**: Python 3.11+, FFmpeg, yt-dlp.
- **AI Engine**: ElevenLabs (Dubbing), Whisper (Transcripts), YOLO/MediaPipe (Auto-Crop).
- **Goal**: Automatically transform long-form (landscape) videos into viral short-form (portrait/9:16) content.
